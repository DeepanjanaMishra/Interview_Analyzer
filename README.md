# Interview_Analyzer - AI-Powered Interview Analysis Platform 🎤🤖

*"Transforming interview practice into data-driven improvement."*

Welcome to **Interview_Analyzer**, an AI-powered interview analysis platform designed to help students, job seekers, and professionals improve their interview performance through intelligent feedback and analytics.

Interview_Analyzer combines modern web technologies with Artificial Intelligence to analyze facial expressions, speech emotions, communication patterns, and interview behavior, turning every mock interview into a measurable learning experience.

---

## 🌟 The Vision

Interviews are often stressful because candidates receive very little objective feedback after practice sessions.

Interview_Analyzer was built with a simple goal:

> **What if candidates could receive instant, AI-driven feedback on their interview performance just like athletes receive performance analytics after a match?**

Instead of relying solely on self-evaluation, users receive insights about:

* Emotional state during the interview
* Speaking confidence
* Communication patterns
* Behavioral trends
* Performance progression over time

---

## ✨ Core Features

### 🎥 Facial Emotion Analysis

Using deep learning and computer vision, Interview_Analyzer analyzes facial expressions during interviews and detects emotions such as:

* Happy
* Neutral
* Sad
* Angry
* Fear
* Surprise
* Disgust

This helps users understand their non-verbal communication patterns.

---

### 🎤 Speech Emotion Recognition

The audio analysis engine processes interview recordings and detects emotions including:

* Calm
* Happy
* Neutral
* Sad
* Angry
* Fearful
* Disgust
* Surprise

---

### 📊 Interview Analytics Dashboard

After every interview session, users receive:

* Emotion analysis results
* Interview history
* Performance tracking
* Session summaries
* Behavioral insights

---

### 📈 Progress Tracking

Users can track improvement across multiple interview sessions:

* Historical interview records
* Previous performance comparison
* Continuous learning feedback loop

---

### 🔐 Secure Authentication

Interview_Analyzer includes:

* User registration
* Login authentication
* JWT-based session management
* Protected routes

## ⚙️ Installation

### Clone Repository

```bash

git clone <repository-url>
cd Project_Code
```

### Install Frontend Dependencies

```bash

cd frontend
npm install
```

### Install Backend Dependencies

```bash

cd ../backend
npm install
```
### Install Python Dependencies

```bash

pip install flask
pip install flask-cors
pip install torch
pip install tensorflow
pip install librosa
pip install pydub
pip install pillow
pip install torchvision
```
---

## ▶️ Running the Project

### 1. Start MongoDB

```bash

mongod
```

### 2. Start Facial Emotion API

```bash

cd facial_emotions
python app.py
```
### 3. Start Audio Emotion API

```bash

cd audio_emotions
python audio_api.py
```

### 4. Start Backend Server

```bash
cd backend
node server.js
```

### 5. Start Frontend

```bash

cd frontend
npm run dev
```
